# Cyber-Safety-Awareness-for-Students
A short educational document promoting safe digital habits for secondary school students.

# Contents
- Email Safety  
- Password Hygiene  
- Safe Browsing Tips  
- Practical References

# Purpose
This mini-project is meant for classroom use or as a public awareness handout.  
You can share it on GitHub, school websites, or print it as a digital safety guide.

# How to Use
1. Clone or download this repository.  
2. Open `Cyber_Safety_Awareness_for_Students.md` in any markdown viewer or browser.  
3. Optionally convert it to PDF or DOCX for distribution.
